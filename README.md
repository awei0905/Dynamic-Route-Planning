# Dynamic-Route-Planning
工科賽動態路徑規劃

## 說明
<p>目前只有找路徑功能，沒有加入動態改變起點、終點及障礙點的功能，也完全沒有提示文字及按鈕。<br />
  找路徑演算法：BFS 廣度優先搜尋<br />
</p>

<p>紅色方塊：起點</p>
<p>藍色圓點：終點</p>
<p>藍色線段：找到從起點至終點的最短路徑</p>
